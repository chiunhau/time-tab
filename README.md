#Time Tracker 
這個chrome extension可以追蹤你在每個網站上花了多少時間

##如何使用開發中版本
1. 點右方下載後解壓縮這份zip 
2. 前往`chrome://extensions/`，勾選「開發人員模式」
3. 勾選後，左邊會出現「載入未封裝功能」，點選它然後找到剛剛解壓縮的專案目錄
4. Enjoy It!!!

##開發中功能
1. 數據視覺化介面（使用D3.js）
2. 效能優化（目前背景很吃資源）
